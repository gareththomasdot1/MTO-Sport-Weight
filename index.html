HELP & INFORMATION

What Does This App Do?

This is a conceptual flight performance calculator for light aircraft. It estimates takeoff distance, rate of climb, and overall flight safety status based on aircraft specifications and current weather conditions.

How It Works

Input Parameters:

  • Pilot/Passenger Weight: Adjust weight sliders (ranges vary by aircraft profile)
  • Baggage & Fuel: Select cargo and fuel load amounts
  • Aircraft Profile: Choose from presets (MTO-Sport, Cavalon models) or custom values
  • Home Airfield: Enter ICAO code to get runway info and weather

Calculation Method

The app calculates performance adjustments using:

  • Weight Factor: Payload impact on takeoff distance (quadratic scaling)
  • Density Altitude: Pressure altitude adjusted for temperature
  • Wind Component: Headwind benefit and tailwind penalty on runway
  • Safety Factor: 30% multiplier on calculated distance for safety margin

Safety Status Indicators

  • ✓ OK - All parameters within normal limits
  • ⚠ CAUTION - Operating within 20% of limits, take care
  • ✗ NOT OK - Operation not recommended (overweight, insufficient runway, poor climb)

Key Limitations

  • This is a simplified model - does not account for all real-world factors
  • Weather data is live but may have delays or inaccuracies
  • Runway information is limited to a database of sample airfields
  • Aircraft performance data are estimates and may differ from actual values
  • NOT authorized or certified for flight planning

Data Sources

  • Weather: Open-Meteo API (real-time atmospheric data)
  • Aircraft Profiles: Manufacturer specifications (simplified)
  • Airfield Data: Manual database (limited)

Always consult official aircraft manuals and regulatory guidance before any flight operation.
